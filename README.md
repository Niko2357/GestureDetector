# Gesture Detector
Using mediapipe library this project detects 21 point in one's hand and matches them with letters according to this diagram.
## Poznámky
* přidat automatické psaní textu, do 3 sekund se to napíše
* přidat čísla, jako další mód
* navíc znaky - mezera, tečka, otazník, zpět
